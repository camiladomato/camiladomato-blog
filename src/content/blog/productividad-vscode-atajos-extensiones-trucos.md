🚀 VS Code puede hacerte programar mucho más rápido

La mayoría usamos VS Code todos los días… pero muy pocos aprovechan realmente todo lo que ofrece.

Después de años trabajando con proyectos React, Angular y Node.js, encontré configuraciones y herramientas que literalmente me ahorran horas por semana.

Acá dejo algunas de las más útiles 👇
⚡ Atajos que te cambian la vida

🔎 Abrir cualquier cosa rápido

CTRL + P

Buscás archivos instantáneamente.

🧠 Paleta de comandos

CTRL + SHIFT + P

Prácticamente podés hacer TODO desde ahí.

✍️ Renombrar variables/componentes

F2

Cambia el nombre en todo el proyecto automáticamente.

📂 Terminal integrada

CTRL + Ñ

Abrís la terminal sin salir del editor.

🖥️ Múltiples cursores

ALT + CLICK

Editás varias líneas al mismo tiempo.

🔥 Extensiones que realmente valen la pena

🎨 Prettier

Formatea automáticamente el código y mantiene consistencia en el proyecto.

✅ ESLint

Detecta errores y malas prácticas mientras escribís.

🏷️ Auto Rename Tag

Renombra etiquetas HTML/JSX automáticamente.

🌳 GitLens

Hace Git muchísimo más visual e intuitivo.

⚡ Error Lens

Muestra errores directamente en línea sin tener que abrir paneles.

🛠️ Configuración simple que mejora muchísimo la experiencia

En settings.json agregá:

{

  "editor.formatOnSave": true,

  "editor.minimap.enabled": false,

  "files.autoSave": "afterDelay",

  "editor.fontLigatures": true

}

✅ Menos clics

💡 Consejo final

No se trata de escribir más código.

Se trata de eliminar fricción y trabajar más cómodo.

Tu editor es tu herramienta principal como developer. Vale muchísimo la pena optimizarlo. 🚀
